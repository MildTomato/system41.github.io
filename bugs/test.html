Report a bug

This utility will allow you to report bugs to the Windows 96 team.

Before submitting a bug, please ensure:

  • any personal information is censored.
  • you are using a clean copy of Windows 96 to rule out any interference from scripts.
  • you are using the correct version of Windows 96.
  • you provide instructions on how to reproduce it.
  • you provide screenshots if possible (use imgur and paste urls in description).

Do not waste our time with nonsense bug reports. You are IP logged and will be banned from submitting future reports.
